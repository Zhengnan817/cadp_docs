Getting Started with the Climate Adaptation Data Platform

7 Building the Platform

The full platform can be built using the following command:

make all

This will first build the platform and then launch it. The following tasks are executed:

  • initialize database
  • initialize the dashboard
  • build all images

You can use this command repeatedly. The Makefile is smart enough to know not to re-initialize the database.

To verify that the system is running, you can check the logs or check the container statuses directly. Logs can be monitored using make logs, while the container statuses can be viewed by typing docker ps. In this latter command, the output should look something like this:
